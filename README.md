# Intro to TypeScript

## Installation and Setup

`npm install -g typescript`

Verify: `tsc`

First `helloWorld.ts`:

```js
console.log("Hello World");

function hello(string: String) {
	console.log("Hello " + string);
}

hello("universe");
hello(5);
```
